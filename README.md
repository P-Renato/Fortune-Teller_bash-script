# Fortune Teller

A fun and simple Bash script that predicts when you'll become rich based on your name and age.

## Description

This script acts as a lighthearted fortune teller. It asks for your name and age, then generates a random future age (between your current age and your current age + 15) when you will supposedly become rich. It's a great example of combining user input, variables, and random number generation in Bash.

## Features

- **Personalized Greeting**: Asks for and uses your name throughout the script
- **Age Input**: Captures your current age
- **Random Fortune**: Generates a random future age (current age + random number between 0-14)
- **Interactive Experience**: Includes a small delay for dramatic effect
- **Lighthearted Output**: Delivers a fun prediction about your financial future

## Requirements

- Bash shell (Linux, macOS, or WSL on Windows)
- Terminal with standard input/output capabilities
